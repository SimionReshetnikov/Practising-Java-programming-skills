# Задание 2: Управление списком студентов 

## Описание

Создать систему для управления списком студентов с использованием `ArrayList`.  

Задание направлено на проверку навыков:
- работы со списками объектов
- сортировки
- поиска
- фильтрации данных

---

## Что нужно сделать

### 1. Создать класс `Student`

**Поля:**
- `id` (`int`) — уникальный идентификатор (генерируется автоматически)
- `name` (`String`) — имя студента
- `grade` (`double`) — средняя оценка (от 0 до 5)

**Требования:**
- Конструктор принимает `name` и `grade`
- `id` генерируется автоматически (использовать статический счетчик)
- Реализовать геттеры и сеттеры
- Переопределить метод `toString()` для вывода информации о студенте
- Переопределить методы `equals(Object o)` и `hashCode()`  
  → студенты считаются равными по `id`

---

### 2. Создать класс `StudentManager`

**Поле:**
- `List<Student> students`

**Конструктор:**
- инициализирует пустой список

---

### 3. Реализовать методы в `StudentManager`

`void addStudent(String name, double grade)`  Добавляет нового студента. Если оценка вне диапазона 0–5 — выбросить `IllegalArgumentException`

`boolean removeStudent(int id)`  Удаляет студента по id. Возвращает `true`, если удаление успешно 

`Student findById(int id)`  Возвращает студента по id или `null`, если не найден 

`List<Student> findByName(String name)`  Возвращает список студентов с указанным именем (без учета регистра) 

`List<Student> getTopStudents(int count)`  Возвращает список из `count` студентов с наивысшими оценками 

`double getAverageGrade()`  Возвращает среднюю оценку всех студентов 

`void sortByGrade()`  Сортирует студентов по оценке (по убыванию) 

`void sortByName()`  Сортирует студентов по имени (в алфавитном порядке) 

`List<Student> getStudentsByGradeRange(double min, double max)` Возвращает студентов с оценками в диапазоне `[min, max]` 

`List<Student> getAllStudents()` Возвращает копию списка студентов 

`void clear()` Удаляет всех студентов 

`int size()`  Возвращает количество студентов 

---

### 4. В методе `main` продемонстрировать работу

- Добавить **7–10 студентов** с разными именами и оценками  
- Выполнить:
  - поиск по `id`
  - поиск по имени  
- Показать:
  - сортировку по оценке
  - сортировку по имени  
- Получить:
  - топ-3 студентов  
  - среднюю оценку  
- Выполнить:
  - фильтрацию по диапазону оценок  
  - удаление студента  
- Проверить:
  - что `getAllStudents()` возвращает **копию списка**  
  (изменение копии не должно влиять на оригинал)
