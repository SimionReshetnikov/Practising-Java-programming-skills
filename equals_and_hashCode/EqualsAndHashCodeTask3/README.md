# Duplicate Detector

Учебный проект для отработки навыков работы с `equals()`, `hashCode()` и коллекциями в Java.

---

## Формулировка задания

### Задание: Детектор дубликатов

**Описание:**
Создай класс `DuplicateDetector`, который помогает находить дубликаты в коллекциях. Это задание проверяет понимание работы `equals()` и `hashCode()` в реальном сценарии.

---

### Класс Person

Создай класс `Person` со следующими полями:

`passportSeries` Серия паспорта (4 цифры) 
`passportNumber` Номер паспорта (6 цифр) 
`fullName` Полное имя 
`birthDate` Дата рождения 

**Требования к Person:**
- Конструктор со всеми полями
- Валидация:
  - Серия и номер паспорта не могут быть `null` или пустыми
  - Серия должна состоять из 4 цифр
  - Номер должен состоять из 6 цифр
  - Имя не может быть `null` или пустым
  - Дата рождения не может быть `null` и не может быть в будущем
- Реализация `equals()` и `hashCode()` люди считаются одинаковыми, если совпадают серия и номер паспорта
- Реализация `toString()`
---

### Класс DuplicateDetector

Создай утилитарный класс `DuplicateDetector` (с приватным конструктором) со следующими статическими методами:

`findDuplicates(List<T> list)` Возвращает список дубликатов (элементы, встречающиеся больше одного раза). Каждый дубликат должен появиться в результате один раз

`getUniqueCount(List<T> list)` Возвращает количество уникальных элементов 

`removeDuplicates(List<T> list)` Возвращает новый список без дубликатов (сохраняя порядок первого вхождения) 

`getFrequencyMap(List<T> list)` Возвращает `Map<T, Integer>`, где ключ — элемент, значение — количество его вхождений в список
