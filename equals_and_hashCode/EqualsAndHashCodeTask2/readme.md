# Student Management System

Учебный проект для отработки навыков работы с `equals()`, `hashCode()` и коллекциями в Java.

---

## Формулировка задания

### Задание: Сравнение студентов (equals и hashCode)

**Описание:**
Создай класс `Student` с полями: `id`, `firstName`, `lastName`, `email`, `grade`.

**Требования:**

1. **Класс `Student`:**
   - Поля: `id`, `firstName`, `lastName`, `email`, `grade`
   - `id` уникальный
   - Конструктор со всеми полями
   - Валидация: `firstName`, `lastName`, `email` не могут быть `null` или пустыми; `grade` от 2.0 до 5.0

2. **Реализация `equals()` и `hashCode()`:**
   - Студенты считаются одинаковыми **только по `id`** (уникальный идентификатор)
   - Даже если у двух студентов одинаковые имя, фамилия, email и оценка, но разные `id` — они разные

3. **Реализация `toString()`:**
   - Формат: `"Student{id=1, name='John Doe', email='john@example.com', grade=4.5}"`

4. **В `main` продемонстрируй:**
   - Создание 5 студентов с разными `id`
   - Попытку добавить студента с уже существующим `id` — он не должен добавиться
   - Вывод всех студентов
   - Проверку `contains()` для существующего и несуществующего студента
   - Изменение оценки у студента и проверку, что он остается в `HashSet` (поиск по `id` должен работать)
